## Ngrok Setup
1. Download [ngrok](https://ngrok.com/download)
2. Run CacheCow locally
3. Open a new terminal and run ```./ngrok http 8080```
4. Verify it's working by sending a ```GET``` request to ```https://<ngrok-forwarding-url>/hello_world```
